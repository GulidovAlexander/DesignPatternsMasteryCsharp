Задание: Реализация Factory Method для создания различных типов транспортных средств
Создайте интерфейс IVehicle с методом Drive(), который будет выводить информацию о типе транспортного средства.

Реализуйте три класса транспортных средств:

Car

Bike

Truck

Каждый из этих классов должен реализовывать интерфейс IVehicle и выводить соответствующую информацию в методе Drive().

Создайте абстрактный класс VehicleFactory с фабричным методом CreateVehicle(), который возвращает объект типа IVehicle.

Реализуйте три класса фабрик:

CarFactory

BikeFactory

TruckFactory

Каждая из этих фабрик должна переопределять метод CreateVehicle() для создания соответствующего типа транспортного средства.

Напишите клиентский код, который будет использовать различные фабрики для создания транспортных средств и вызывать метод Drive() для каждого из них.

Driving a Bike
Driving a Truck
